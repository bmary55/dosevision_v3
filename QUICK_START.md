# DoseVision - Quick Start Guide

## 3-Step Deployment

### Step 1: Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/dosevision.git
cd dosevision
```

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Run the Application
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Dose Ordering Optimization**: Smart financial ordering and vendor management
- **Regulatory Compliance**: 9 compliance modules for medical imaging
- **Professional UI**: Built with shadcn/ui and Tailwind CSS
- **Excel Export**: Export data to Excel format
- **Real-time Updates**: Live data synchronization

## Technology Stack

- Next.js 15.5.3
- React 19.1.0
- TypeScript 5
- Tailwind CSS 4
- Prisma ORM
